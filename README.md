# HackerRank Solutions (Java)

This repository contains my solutions to various problems from HackerRank, all implemented in Java.

## Purpose

The goal of this repository is to:

* Practice problem-solving skills
* Improve understanding of data structures and algorithms
* Keep a structured record of solved problems

## Structure

Problems are organized based on their domain or topic, such as:

* Algorithms
* Data Structures
* Java
* Problem Solving

Each solution file includes:

* Problem name (or a close reference)
* Clean and readable Java code
* Comments where necessary for understanding

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/rishikapandagre/HackerRank.git
   ```

2. Navigate to the project folder:

   ```bash
   cd HackerRank
   ```

3. Open files in your preferred IDE (e.g., IntelliJ IDEA, Eclipse, VS Code)

4. Run any Java file to test the solution

## Requirements

* Java 8 or above
* Basic understanding of Java programming

## Notes

* Solutions are written with clarity in mind, not just optimization
* Some problems may have multiple approaches; only one is implemented unless stated
* File naming may vary slightly from original problem titles
